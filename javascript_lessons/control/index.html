構造化プログラミング　→　プルグラムの構造は大きく3つに分類される

  • 順次　→　記述された順番に処理を行っていく
  • 選択　→　条件によって処理を分岐する
  • 反復　→　特定の処理を繰り返し実行する

条件によって処理を分岐する

  • if
  • switch
繰り返し処理/反復処理
  • for
  • for ... in
  • for ... of
  • while
  • do ... while

whileとdo ... whileの違い

  • while　→　ループの最初で条件式を判定する（前置判定）
  • do ... while　→　ループの最後で条件式を判定する（後置判定）
